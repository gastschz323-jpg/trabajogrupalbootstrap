## Descripción
Sitio web informativo sobre la Ciudad de Mendoza que incluye una sección de inicio con la presentación de la ciudad,
información sobre su historia, sus principales puntos turísticos y su gastronomía.
## Componentes de Bootstrap utilizados
- Navbar responsivo
- Carousel (carrusel de imágenes)
- Sistema de grillas (Grid)
- Cards
- Accordion
- Alerts
- Blockquote
